load_dataset(lines)¶

Read string-like facts into a dictionary object.

Parameters

lines (list of strings) – a list of facts in the form of A(x,y,z)@[1,2] or A@[1,2)

Returns

A defaultdict object, in which the key is the predicate and the value is a dictionary (key is the entity and the value is a list of Interval instances) or a list of Interval instance when there is no entity.
